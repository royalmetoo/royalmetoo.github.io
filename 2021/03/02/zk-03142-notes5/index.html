[03142互联网及其应用]笔记-第5章 网络应用制作技术

张小五 作于 2021年03月02日

HTML

HTML超文本标记语言是一种计算机程序语言，专门用来编写网页。
HTML文件是层次结构文件，它分为文件头和文件体两部分。文件头包含与文件有关的信息，文件体包含全部正文信息。
网页设计原则有如下几点：

  • 符合用户的需求。
  • 有效使用资源。
  • 生成一个一致的、令人愉快的、有效的网页外观。
  • 在制作网页之前，应明确制作目的、需完成的任务和要达到的要求，应尽可能地收集素材。
  • 网页设计要有自己的风格，通常应有一定的标志。
  • 网页页面不必太花哨，图片不宜太多，图片不应太大。
  • 网页的制作要尽量使其通用。

网页的设计方法：

  • 使用任意的文本编辑软件，在其中写入HTML的各种标记及其属性参数，并以.htm或.html后缀保存。
  • 使用工具软件设计网页，比如Dreamweaver等。

Flash的特点是矢量图像、交互性、流技术。

JavaScript

JavaScript是一种基于对象和事件驱动、并具有安全性的脚本语言。JavaScript是一种基于客户端浏览器的语言。JavaScript的特点是简单性、动态性、跨平台性。
在HTML代码中，JavaScript代码用<script language=”JavaScript”>和</script>标识括起来，可以任意分部于<HEAD>标识和<BODY>标识之中。
JavaScript有6种数据类型：string类型、number类型、Boolean类型、Object对象类型、Null值、Undefined值。
JavaScript变量的名称长度是任意的，区分大小写，需遵循3个规则：第一个字符必须是字母、下划线、$符号之一；后续字符是字母、数字、下划线、$符号的任意组合；非JavaScript的关键字。
表达式是变量、常量和运算符的集合。表达式可以分为算术表达式、字符串表达式和布尔表达式。
运算符是完成操作的一系列符号，JavaScript中有算术运算符、字符串运算符、比较运算符和布尔运算符等。
JavaScript主要有以下事件：

  • 单击事件onClick: 单击按钮、复选框和单选按钮等对象触发。
  • 改变事件onChange: 单选文本框或多选文本框中输入的字符值改变时、下拉列表的选项改变时触发。
  • 选中事件onSelect: 单选文本框或多选文本框中的文字被选中时触发。
  • 获得焦点事件onFocus: 用户单击单行文本框、多行文本框、下拉列表时触发。
  • 失去焦点事件onBlur: 单行文本框、多行文本框、下拉列表失去焦点时触发。
  • 载入文件事件onLoad: 文档载入时触发。
  • 卸载文件事件onUnload: 页面退出时触发。
  • 鼠标悬停事件onMouseover: 鼠标悬停时触发。

XML

XML是可扩展标记语言，是SGML的一个优化子集。XML是一种元标注语言，该语言提供一种描述结构数据的格式。
XML和HTML的区别：HTML是用来表示信息布局的，而XML是用来描述信息本身的；HTML将数据和显示混在一起，而XML则将数据和显示分开；HTML标识不区分大小写，而XML标识区分大小写；HTML可以没有结束标识，而XML标识必须成对出现。
XML文档就是用XML标记书写的XML源代码文件。XML文档包含3部分：一个XML文档声明；一个关于文档类型的定义；用XML标识创建的内容。
XML文档有两种方式向浏览器提供样式表的信息：级联样式表CSS、可扩展样式表XSL。
XSL是一种可以将XML转化为HTML的语言，一种可以过滤和选择XML数据的语言，一种能够格式化XML数据的语言，它包含3部分：一个转换XML文档的方法；一个定义XML部分和模板的方法；一个格式化XML文档的方法。XLS文档本身也是一个XML文档。

动态页面技术

分布式数据库是指物理上分布、逻辑上系统结构集中的数据库。
实现Web数据库一般有两种方法：一种是在Web服务器端提供中间件来连接Web服务和数据库服务器；另一种是把应用程序下载到客户端，并在客户端直接访问数据库。
中间件负责管理Web服务器和数据库服务器之间的通信并提供应用程序服务，它能够直接或调用外部程序或脚本代码来访问数据库，因此可以提供与数据库相关的动态HTML页面，或执行用户查询，并将查询结果格式化成HTML页面，通过Web服务器返回给用户浏览器。最基本的中间件技术有**通用网关接口(CGI)和应用程序编程接口(API)**。

ASP是一个服务器端执行的程序，能够产生和执行动态的、交互的、高效的Web服务器的应用程序，其特点有：

  • 集成于HTML中，无须编译或链接就能执行。
  • 使用常规文档编辑器就可以设计ASP程序。
  • 与浏览器无关。
  • 源代码不会传送到浏览器，更安全。
  • 可通过ActiveX服务器元件来扩充功能。
  • 面向对象。

ASP提供了5个内置对象：Response对象、Request对象、Server对象、Application对象、Session对象。
Response对象用于传送信息给用户，将程序执行的结果传送给浏览器。

  • Response.write: 把信息写在浏览器上。
  • Response.redirect: 可以将目前的网页转移到另一个网页。
  • Response.end: 告诉浏览器信息已经全部下载完毕。

Request对象用于读取浏览器的信息。

Server对象提供访问服务器的方法和属性，是VB Script程序建立ActiveX对象的源头。

  • Server.MapPath: 把“网址路径”转换成服务器的“实际路径”。
  • Server.HTMLEncode: 对特殊字符如“<”及“>”进行编码。
  • Server.URLEncode: 对空格和汉字进行编码。

Application对象提供多个网页之间保留和使用的一些公用信息，它被访问该网站的所有用户共享。

Session对象也是ASP文件公用的对象，提供网页之间的公用信息，但是它只能被一个连接者访问，每个连接者都拥有自己的Session对象。

ASP访问数据库是通过ADO对象实现的。当服务器解释到ADO对象时，会调用相应的数据库驱动程序来访问数据库中的资源，并把访问结果返回给浏览器。

PHP超文本预处理器，是一种在服务端执行的嵌入HTML的脚本语言。PHP是源代码开放程序，拥有很好的跨平台兼容性。

JSP是Sun公司推出的新一代站点开发语言，可以在Servlet和JavaBean的支持下完成功能强大的站点。
JSP的技术特点如下：

  • 内容的生成和显示进行分离。
  • 生成可重用的组件。
  • 采用标识简化页面开发。
  • 具有Java技术的所有优点。
  • JSP可以“一次编写，各处运行”。
